# Mini-Pipeline for Discovery and Prioritization of Cell-Type-Specific Biomarkers in Human Liver Fibrosis

## Overview

This project analyzes publicly available human liver single-cell RNA-seq data from GSE136103 to identify fibrosis-associated cell populations, disease-linked cellular states, and translational biomarker candidates for human liver cirrhosis.

The analysis focuses on:

- Hepatic stellate / fibroblast-like cells
- Myeloid / macrophage populations
- Endothelial cells
- Fibrosis-associated gene signatures
- Biomarker prioritization

## Dataset

Primary dataset:

- GSE136103: Resolving the fibrotic niche of human liver cirrhosis using single-cell transcriptomics

The raw matrix files were downloaded from GEO and processed locally. Raw data files are not included in this repository due to file size.

Methods

The workflow included:

1. Loading raw 10x-style matrix, barcode, and gene files
2. Metadata curation for healthy and cirrhotic liver samples
3. Quality control using detected genes, total counts, and mitochondrial percentage
4. Normalization and log transformation
5. Highly variable gene selection
6. PCA, neighborhood graph construction, UMAP, and Leiden clustering
7. Cell-type annotation using canonical marker genes
8. Cell composition analysis by disease condition
9. Differential expression analysis within each cell type
10. Stellate-cell subclustering
11. Pathway enrichment analysis
12. Rule-based biomarker prioritization

Main Results

After QC, 60,455 cells were retained for downstream analysis.

Annotated cell populations included:

* T cells
* NK cells
* Myeloid cells
* Endothelial cells
* Hepatocytes
* Cholangiocytes
* Plasma cells
* Fibroblast/Stellate cells
* Mast cells

A cirrhosis-associated activated stellate-cell state was identified. This population represented approximately 40.1% of cirrhotic stellate cells compared with 3.7% of healthy stellate cells.

Key fibrosis-associated genes included:

* TIMP1
* TGM2
* CST3
* EFEMP1
* LGALS1
* CCL21
* IL32
* IFITM2
* IFITM3
* RAMP2

Pathway Analysis

Pathway enrichment of activated stellate-cell genes identified enrichment for:

* Cytokine response
* Type I and Type II interferon response
* Peptide cross-linking
* Protease regulation
* Receptor internalization
* Inflammatory stromal remodeling

Biomarker Prioritization

A rule-based prioritization score was used to rank candidate biomarkers based on:

* Differential expression strength
* Fibrosis relevance
* Translational potential
* Cell-state specificity

Top-ranked candidates:

1. TIMP1
2. TGM2
3. CST3
4. EFEMP1
5. LGALS1
6. CCL21
7. IL32
8. IFITM2
9. IFITM3
10. RAMP2
